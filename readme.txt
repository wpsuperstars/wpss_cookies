== Description ==

![alt WordPress Cookie Consent Plugin](assets/images/plugin_logo.jpg)

== Installation ==
1. Upload plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy
